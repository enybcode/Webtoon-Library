## Resume

Webtoon-Library est une application PHP pour gerer une bibliotheque personnelle de webtoons, manhwa et mangas.

L'utilisateur peut se connecter, rechercher des oeuvres via AniList, les ajouter a sa liste, consulter une page detail et suivre sa lecture avec un statut, un chapitre, une note et un commentaire.

Le projet utilise PHP, PDO, sessions, AniList GraphQL et une interface admin simple.
